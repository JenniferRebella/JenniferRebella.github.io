### Artifact:
I decided to use just one artifact to meet all requirements. My artifact is my Inventory App from my final project for my course CS-360. This Android application connects to a MongoDB database and only allows users to create one list, and does not have a functioning login screen. To meet all the requirements, I decided to use my skills in full-stack development and create a port to Apple's iOS by using Swift and swift UI in development, which I have never used before, to meet the software design and engineering requirements. To meet the algorithms and data structure requirements, I added a search bar for each of the users' lists that allows for the user to search for a spefic item based on name or ID. Lastly, for the database portion, I decided to use a new database manager and use Google's Firebase as opposed to MongoDB, as Firebase is a better fit for swift code than MongoDB, as well as adding another list for users to add items to.
